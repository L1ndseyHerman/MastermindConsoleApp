## Welcome!

To view this app starting on GitHub, please go to the Code button, the Local tab,
and then Open with Visual Studio. If you're on a Mac... uh... good luck.
I know some people at my old job used Ryder for their C# on Mac needs.
Maybe you can run this in there, idk....

Anyways, for Visual Studio, start by opening the solution. Once it's booted up,
make sure Debug, AnyCPU, and MastermindConsoleApp are what's selected
in the dropdowns at the top. Once those are selected, click the play button
on the MasterMindConsoleApp. From there, a command-line window should pop up
with instructions on how to play the game.
