## Welcome!

To view this app starting on GitHub, please go to the Code button, the Local tab,
and then Open with Visual Studio. If you're on a Mac... uh... good luck.
I know some people at my old job used Ryder for their C# on Mac needs.
Maybe you can run this in there, idk....

Anyways, for Visual Studio, start by opening the solution. Once it's booted up,
make sure Debug, AnyCPU, and MastermindConsoleApp are what's selected
in the dropdowns at the top. Once those are selected,
click the filled-in play button on the MasterMindConsoleApp.
From there, a command-line window should pop up
with instructions on how to play the game.

(I know ppl at the company I am interviewing with already have the instructions,
I'm just trying to show I know how to write ReadMes and instructions
for cases where ppl don't know :))

To run the XUnit tests, go to the "Test" button at the top, click on it,
then scroll down to "Test Explorer" and click on that.
A Test Explorer view should open.
On it, make sure to select the beaker icon for all the tests at the top
and the outermost MastermindConsoleAppTests on the table.
Then click the top filled-in play button right below "Test Explorer"
to run the tests and they should all pass :)
